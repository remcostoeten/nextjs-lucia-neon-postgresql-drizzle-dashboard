# UI Import Consolidator

Replaces multiple '@/components/ui' imports:

```
import { Button } from '@/components/ui/button'
import { Card } from '@/components/ui/card'
```

With a single import:

```
import { Button, Card } from 'ui'
```

Requires:

-   index.ts in ui folder exporting all components
-   'ui' alias in tsconfig.json linking to the index.ts file in components/ui
    Run:

```
python consolidate_imports.py --folder src/app/feature
```
